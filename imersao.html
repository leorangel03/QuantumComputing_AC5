Como ingressar na área

Atualmente não há um mercado consolidado para área da computação quântica entretanto com seu acelerado desenvolvimento, em um futuro próximo há grandes possiblidades de surgir oportunidades profissionais nesta área. E para isso é imprescindível formações que te ajudaram a agarrar essas oportunidades, então nessa página iremos instruí-lo a quais instituiçoes e cursos você deve fazer, além disso iremos propor um guia passo a passo para você começar a aprender!

No Brasil, ainda não existem muitas opções de cursos de graduação e pós-graduação em computação quântica, mas há algumas instituições que oferecem programas de formação nessa área ou até cursos livres que se tornam muito efecientes.

“Havia muitas promessas, houve alguns avanços experimentais, mas nada que trouxesse um grande otimismo de que iríamos ter um computador quântico funcional resolvendo alguma coisa importante. Faltava um marco, mas ele chegou em 2019, com o Google.
Com esse novo impulso, as empresas passaram a mirar a computação quântica porque enxergam nela um caminho inevitável de transformação dos negócios."

Projeto recomendado

Apresentando a ferramenta: O Quantum Information Software Development Kit (Qiskit) é uma estrutura computacional de código aberto desenvolvida para funcionar em diferentes linguagens de programação como Python, Swift e JavaScript, fornecendo as ferramentas necessárias para a criação de algoritmos quânticos, seguindo um modelo de circuito para computação quântica universal, e a sua execução em dispositivos quânticos reais usando o acesso remoto aos hardwares disponibilizados através do IBMQ. Além do Qiskit voce irá utilizar a linguagem de programação Python, e um software para o desenvolvimento do algoritmo (recomendamos o Jupyner juntamente ao Anaconda)

Descrição do projeto: Fornecer uma breve introdução aos conceitos fundamentais de computação quântica, usando os ambientes computacionais do Qiskit. Desvendando os conceitos de qubits, emaranhamento quântico, circuitos e algoritmos.

A seguir temos a imagem do resultado final que é a execução do circuito quantico desenvolvido, em um computador clássico (simulando um quantico pelo Qiskit AER) e ver o que acontece quando simulamos em um computador quântico real.

Experiências práticas

Nessa experiencia iremos desenvolver o circuito quantico referente aos resultados do projeto recomendado anteriormente.

Para realizarmos o desenvolvimento desse circuito é necessário o uso da API do IBMQ ("computador quantico virtual" forneceido pela IBM para fundos academicos), para voce adquirir o valor do token da API acesse: https://quantum-computing.ibm.com/account.

Além disso como foi dito anteriormente iremos utilizar o software livre Jupyter Notebook juntamente do Anaconda em Python (linguagem projetada para ser de fácil leitura, com rápido desenvolvimento de código e de fácil compreensão) para o desenvolvimento da experiencia.

Nas imagens a seguir iremos ver todo o desenvolvimento da experiencia do circuito, com as celulas de códigos já executadas.
